# 安裝&載入Package

## 安裝套件指令

```r
install.packages("套件名稱")
```

{% hint style="info" %}
套件名稱要加上雙引號 " "
{% endhint %}

## 載入套件

```r
library(套件名稱)
```

## 常用套件

```r
install.packages("rJava")
install.packages("xlsxjars")
install.packages("xlsx")
install.packages("data.table")
install.packages("plyr")

```
